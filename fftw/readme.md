### Program Usage
```
leiming@homedesktop:~/git/cuda_fft/fftw$ ./fftw 
[LOG] Signal Length: 4096
[LOG] FFT run: 10000
[LOG] Trials: 10

leiming@homedesktop:~/git/cuda_fft/fftw$ ./fftw -l 1024 -r 1024 -t 20
[LOG] Signal Length: 1024
[LOG] FFT run: 1024
[LOG] Trials: 20
```
